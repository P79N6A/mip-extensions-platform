# mip-hs-tabs

mip-hs-tabs 组件说明
此组件可用于移动滑动导航，高亮页面状态自动剧中。
标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-hs-tabs/mip-hs-tabs.js
