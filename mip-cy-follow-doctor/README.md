# mip-cy-follow-doctor

mip-cy-follow-doctor组件实现用户关注春雨医生～

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/extensions/platform/v1/mip-cy-follow-doctor/mip-cy-follow-doctor.js

## 示例

### 基本用法
```html
<mip-cy-follow-doctor
    doctorId="xxxxx"
></mip-cy-follow-doctor>
```

## 属性

### doctorId

说明：关注的医生的doctorId
必选项：是
类型：字符串
取值范围：无
单位：无
默认值：''

### auto

说明：是否自动调取关注医生
必选项：否
类型：数字
取值范围：非负整数
单位：无
默认值：0
