# mip-666-addclass

mip-select-addclass 点击包含指定class的标签，切换增删class样式

标题|内容
----|----
类型|通用
所需脚本|https://c.mipcdn.com/static/v1/mip-666-addclass/mip-666-addclass.js



## 示例

### 基本用法
```html
<mip-666-addclass selector=".menu>li" toggleclass="active">
 	<ul class="menu" >
         <li class="active">xxx</li>
         <li>xxx</li>
         <li>xxx</li>
         <li>xxx</li>
	</ul>
</mip-666-addclass>
```



## 属性

### selector

说明：选择器
必选项：是
类型：字符串

### toggleclass

说明：切换样式class
必选项：是
类型：字符串
