# mip-connect

mip-connect 锚链接

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-connect/mip-connect.js

## 示例

### 基本用法
```html
<mip-connect>
	<section id="link_mlj">
		<li>本周热门项目</li>
		<li>最新上架项目</li>
		<li>本月精选品牌</li>
	</section>
	<div class="link_main">
		<div class="mlj">第一段链接内容</div>
		<div class="mlj">第二段链接内容</div>
		<div class="mlj">第三段链接内容</div>
	</div>
	<p></p>
</mip-connect>

```

## 属性

### {cObj:该组件对象class}
### {lBox:跳转的对象}

naver(cObj, lBox);

说明：{锚链接}
必选项：{是}
类型：{responsive}

默认值：{默认值}
