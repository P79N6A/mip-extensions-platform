# mip-yp-rem

mip-yp-rem rem换算

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-yp-rem/mip-yp-rem.js
## 示例

### rem换算
```html
<mip-yp-rem></mip-yp-rem>
```

### 自定义参数

```html
<mip-alert remV="6.4">
    自定义内容，可以嵌套其他组件
</mip-alert>
```
